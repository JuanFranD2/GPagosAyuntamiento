◆ page_script()

page_script ( $callback)

Processes a callback on every page.

The callback function receives the four parameters int $pageNumber, int $pageCount, Canvas $canvas, and FontMetrics $fontMetrics, in that order.

This function can be used to add page numbers to all pages after the first one, for example.

callable$callbackThe callback function to process on every page
