getPrecision ( )

Returns the number of significant digits in the number.

This is the number of digits to both sides of the decimal point, stripped of leading zeros. The sign has no impact on the result.

Examples: 0 => 0 0.0 => 0 123 => 3 123.456 => 6 0.00123 => 3 0.0012300 => 5
