Detailed Description

This class offers convenience methods for parsing and serializing HTML5. It is roughly designed to mirror the \DOMDocument native class.

◆ load()

load ( $file,
array $options = array() )

Load and parse an HTML file.

This will apply the HTML5 parser, which is tolerant of many varieties of HTML, including XHTML 1, HTML 4, and well-formed HTML

  1. Note that in these cases, not all of the old data will be preserved. For example, XHTML's XML declaration will be removed.

The rules governing parsing are set out in the HTML 5 spec.

Parameters
string | resource$fileThe path to the file to parse. If this is a resource, it is assumed to be an open stream whose pointer is set to the first byte of input.
array$optionsConfiguration options when parsing the HTML.
Returns
