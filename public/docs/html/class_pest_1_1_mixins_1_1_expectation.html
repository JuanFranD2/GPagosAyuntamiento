◆ toEqualCanonicalizing()

toEqualCanonicalizing ( mixed $expected,
string $message = '' )

Asserts that two variables have the same value. The contents of $expected and the $this->value are canonicalized before they are compared. For instance, when the two variables $expected and $this->value are arrays, then these arrays are sorted before they are compared. When $expected and $this->value are objects, each object is converted to an array containing all private, protected and public attributes.

Returns
