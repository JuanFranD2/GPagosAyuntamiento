◆ __set()

__set ( string $prop,
$val )

Set a style property to its final value.

This sets the specified and used value of the style property to the given value, meaning the value is not parsed and thus should have a type compatible with the property.

If a shorthand property is specified, all of its sub-properties are set to the given value.
