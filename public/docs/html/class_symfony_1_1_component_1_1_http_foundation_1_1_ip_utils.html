◆ anonymize()

anonymize ( string $ip)
static

Anonymizes an IP/IPv6.

Removes the last bytes of IPv4 and IPv6 addresses (1 byte for IPv4 and 8 bytes for IPv6 by default).

Parameters
int<0,4>$v4Bytes
int<0,16>$v6Bytes

If the IP contains a % symbol, then it is a local-link address with scoping according to RFC 4007 In that case, we only care about the part before the % symbol, as the following functions, can only work with the IP address itself. As the scope can leak information (containing interface name), we do not want to include it in our anonymized IP data.
