Detailed Description

Build a callable object, i.e. function, class::method, object->method or closure. They all have the same behaviour. This callable is an extension of native PHP callable (aka callback) to integrate Hoa's structures.

Constructor & Destructor Documentation

◆ __construct()

__construct ( $call,
$able = '' )

Allocates a xcallable based on a callback.

Accepted forms:

  • ‘'function’,
  • 'class::method',
  • 'class', 'method',
  • $object, 'method',
  • $object, '',
  • function (…) { … },
  • ['class', 'method'],
  • [$object, 'method']`.

Exceptions

A Hoa\Consistency\Exception exception is thrown if the callback form is invalid.
