resolve ( $file,
$startLine = null )
static

Runtime validation that $file can be resolved as an include path.

If $file can be resolved, return $file. Otherwise throw a fatal error exception.

If $file collides with a path in the currently running PsySH phar, it will be resolved relative to the include path, to prevent PHP from grabbing the phar version of the file.

Exceptions
FatalErrorExceptionwhen unable to resolve include path for $file
ErrorExceptionif $file is empty and E_WARNING is included in error_reporting level
