\hypertarget{README.md_autotoc_md8682}{}\doxysubsection{\texorpdfstring{Introduction}{Introduction}}\label{README.md_autotoc_md8682}
The Reflection\+Doc\+Block component of php\+Documentor provides a Doc\+Block parser that is 100\% compatible with the \href{http://phpdoc.org/docs/latest}{\texttt{PHPDoc standard}}.

With this component, a library can provide support for annotations via Doc\+Blocks or otherwise retrieve information that is embedded in a Doc\+Block.

\begin{quote}
{\bfseries{Note}}\+: {\itshape this is a core component of php\+Documentor and is constantly being optimized for performance.} \end{quote}

The Reflection\+Doc\+Block component is designed to work in an identical fashion to PHP\textquotesingle{}s own Reflection extension (\href{http://php.net/manual/en/book.reflection.php}{\texttt{http\+://php.\+net/manual/en/book.\+reflection.\+php}}).

Parsing can be initiated by instantiating the {\ttfamily \textbackslash{}php\+Documentor\textbackslash{}Reflection\textbackslash{}Doc\+Block()} class and passing it a string containing a Doc\+Block (including asterisks) or by passing an object supporting the {\ttfamily get\+Doc\+Comment()} method.

\begin{quote}
{\itshape Examples of objects having the {\ttfamily get\+Doc\+Comment()} method are the {\ttfamily Reflection\+Class} and the {\ttfamily Reflection\+Method} classes of the PHP Reflection extension} \end{quote}


Example\+: \begin{DoxyVerb}$class = new ReflectionClass('MyClass');
$phpdoc = new \phpDocumentor\Reflection\DocBlock($class);
\end{DoxyVerb}


or \begin{DoxyVerb}$docblock = <<<DOCBLOCK
/**
 * This is a short description.
 *
 * This is a *long* description.

 * @return void
 */

$phpdoc = new \phpDocumentor\Reflection\DocBlock($docblock);
\end{DoxyVerb}
