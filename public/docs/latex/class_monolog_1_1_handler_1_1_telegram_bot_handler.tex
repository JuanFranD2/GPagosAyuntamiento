\doxysubsection{Detailed Description}
Handler sends logs to Telegram using Telegram Bot API.

How to use\+: 1) Create a Telegram bot with \href{https://telegram.me/BotFather;}{\texttt{https\+://telegram.\+me/\+Bot\+Father;}} 2) Create a Telegram channel or a group where logs will be recorded; 3) Add the created bot from step 1 to the created channel/group from step 2.

In order to create an instance of Telegram\+Bot\+Handler use
\begin{DoxyEnumerate}
\item The Telegram bot API key from step 1
\item The channel name with the {\ttfamily @} prefix if you created a public channel (e.\+g. {\ttfamily @my\+\_\+public\+\_\+channel}), or the channel ID with the {\ttfamily -\/100} prefix if you created a private channel (e.\+g. {\ttfamily -\/1001234567890}), or the group ID from step 2 (e.\+g. {\ttfamily -\/1234567890}).
\end{DoxyEnumerate}
