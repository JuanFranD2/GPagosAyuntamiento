\doxysubsection{Detailed Description}
Value Object representing a Compound Type.

A Intersection Type is not so much a special keyword or object reference but is a series of Types that are separated using an AND operator ({\ttfamily \&}). This combination of types signifies that whatever is associated with this Intersection type may contain a value with any of the given types.
