\doxysubsection{Detailed Description}
This class represents rule sets for generic at-\/rules which are not covered by specific classes, i.\+e., not {\ttfamily @import}, {\ttfamily @charset} or {\ttfamily @media}.

A common example for this is {\ttfamily @font-\/face}.
