assert\+That (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [static]}}

Make an assertion and throw {\ttfamily \doxylink{}{Hamcrest\textbackslash{}\+Assertion\+Error}} if it fails.

The first parameter may optionally be a string identifying the assertion to be included in the failure message.

If the third parameter is not a matcher it is passed to {\ttfamily \doxylink{}{Hamcrest\textbackslash{}\+Core\textbackslash{}\+Is\+Equal\#equal\+To}} to create one.

Example\+: 
\begin{DoxyPre}
// With an identifier
assertThat("{}apple flavour"{}, \$apple-\/>flavour(), equalTo("{}tasty"{}));
// Without an identifier
assertThat(\$apple-\/>flavour(), equalTo("{}tasty"{}));
// Evaluating a boolean expression
assertThat("{}some error"{}, \$a > \$b);
assertThat(\$a > \$b);
\end{DoxyPre}
