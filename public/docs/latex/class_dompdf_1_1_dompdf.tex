set\+Callbacks (\begin{DoxyParamCaption}\item[{array}]{\$callbacks}{}\end{DoxyParamCaption})}

Define callbacks that allow modifying the document during render.

The callbacks array should contain arrays with {\ttfamily event} set to a callback event name and {\ttfamily f} set to a function or any other callable.

The available callback events are\+:
\begin{DoxyItemize}
\item {\ttfamily begin\+\_\+page\+\_\+reflow}\+: called before page reflow
\item {\ttfamily begin\+\_\+frame}\+: called before a frame is rendered
\item {\ttfamily end\+\_\+frame}\+: called after frame rendering is complete
\item {\ttfamily begin\+\_\+page\+\_\+render}\+: called before a page is rendered
\item {\ttfamily end\+\_\+page\+\_\+render}\+: called after page rendering is complete
\item {\ttfamily end\+\_\+document}\+: called for every page after rendering is complete
\end{DoxyItemize}

The function {\ttfamily f} receives three arguments {\ttfamily Frame \$frame}, {\ttfamily Canvas \$canvas}, and {\ttfamily Font\+Metrics \$font\+Metrics} for all events but {\ttfamily end\+\_\+document}. For {\ttfamily end\+\_\+document}, the function receives four arguments {\ttfamily int \$page\+Number}, {\ttfamily int \$page\+Count}, {\ttfamily Canvas \$canvas}, and {\ttfamily Font\+Metrics \$font\+Metrics} instead.
