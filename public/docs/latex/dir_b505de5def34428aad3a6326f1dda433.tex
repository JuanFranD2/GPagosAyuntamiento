Human-\/friendly process signals.

This is a map of known process signals with some information about each signal.

Unlike \href{https://nodejs.org/api/os.html\#os_signal_constants}{\texttt{{\ttfamily os.\+constants.\+signals}}} this includes\+:


\begin{DoxyItemize}
\item human-\/friendly descriptions
\item default actions, including whether they can be prevented
\item whether the signal is supported by the current OS
