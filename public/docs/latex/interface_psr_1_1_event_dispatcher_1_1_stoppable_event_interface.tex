\doxysubsection{Detailed Description}
An \doxylink{class_event}{Event} whose processing may be interrupted when the event has been handled.

A Dispatcher implementation MUST check to determine if an \doxylink{class_event}{Event} is marked as stopped after each listener is called. If it is then it should return immediately without calling any further Listeners.
