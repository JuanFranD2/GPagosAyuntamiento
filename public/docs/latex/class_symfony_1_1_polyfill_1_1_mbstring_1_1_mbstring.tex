\doxysubsection{Detailed Description}
Partial mbstring implementation in PHP, iconv based, UTF-\/8 centric.

Implemented\+:
\begin{DoxyItemize}
\item mb\+\_\+chr -\/ Returns a specific character from its Unicode code point
\item mb\+\_\+convert\+\_\+encoding -\/ Convert character encoding
\item mb\+\_\+convert\+\_\+variables -\/ Convert character code in variable(s)
\item mb\+\_\+decode\+\_\+mimeheader -\/ Decode string in MIME header field
\item mb\+\_\+encode\+\_\+mimeheader -\/ Encode string for MIME header XXX NATIVE IMPLEMENTATION IS REALLY BUGGED
\item mb\+\_\+decode\+\_\+numericentity -\/ Decode HTML numeric string reference to character
\item mb\+\_\+encode\+\_\+numericentity -\/ Encode character to HTML numeric string reference
\item mb\+\_\+convert\+\_\+case -\/ Perform case folding on a string
\item mb\+\_\+detect\+\_\+encoding -\/ Detect character encoding
\item mb\+\_\+get\+\_\+info -\/ Get internal settings of mbstring
\item mb\+\_\+http\+\_\+input -\/ Detect HTTP input character encoding
\item mb\+\_\+http\+\_\+output -\/ Set/\+Get HTTP output character encoding
\item mb\+\_\+internal\+\_\+encoding -\/ Set/\+Get internal character encoding
\item mb\+\_\+list\+\_\+encodings -\/ Returns an array of all supported encodings
\item mb\+\_\+ord -\/ Returns the Unicode code point of a character
\item mb\+\_\+output\+\_\+handler -\/ Callback function converts character encoding in output buffer
\item mb\+\_\+scrub -\/ Replaces ill-\/formed byte sequences with substitute characters
\item mb\+\_\+strlen -\/ Get string length
\item mb\+\_\+strpos -\/ Find position of first occurrence of string in a string
\item mb\+\_\+strrpos -\/ Find position of last occurrence of a string in a string
\item mb\+\_\+str\+\_\+split -\/ Convert a string to an array
\item mb\+\_\+strtolower -\/ Make a string lowercase
\item mb\+\_\+strtoupper -\/ Make a string uppercase
\item mb\+\_\+substitute\+\_\+character -\/ Set/\+Get substitution character
\item mb\+\_\+substr -\/ Get part of string
\item mb\+\_\+stripos -\/ Finds position of first occurrence of a string within another, case insensitive
\item mb\+\_\+stristr -\/ Finds first occurrence of a string within another, case insensitive
\item mb\+\_\+strrchr -\/ Finds the last occurrence of a character in a string within another
\item mb\+\_\+strrichr -\/ Finds the last occurrence of a character in a string within another, case insensitive
\item mb\+\_\+strripos -\/ Finds position of last occurrence of a string within another, case insensitive
\item mb\+\_\+strstr -\/ Finds first occurrence of a string within another
\item mb\+\_\+strwidth -\/ Return width of string
\item mb\+\_\+substr\+\_\+count -\/ Count the number of substring occurrences
\item mb\+\_\+ucfirst -\/ Make a string\textquotesingle{}s first character uppercase
\item mb\+\_\+lcfirst -\/ Make a string\textquotesingle{}s first character lowercase
\item mb\+\_\+trim -\/ Strip whitespace (or other characters) from the beginning and end of a string
\item mb\+\_\+ltrim -\/ Strip whitespace (or other characters) from the beginning of a string
\item mb\+\_\+rtrim -\/ Strip whitespace (or other characters) from the end of a string
\end{DoxyItemize}
