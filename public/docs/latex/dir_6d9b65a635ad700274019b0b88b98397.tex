\DoxyCodeLine{use\ \mbox{\hyperlink{class_barryvdh_1_1_dom_p_d_f_1_1_facade_1_1_pdf}{Barryvdh\(\backslash\)DomPDF\(\backslash\)Facade\(\backslash\)Pdf}};}

\end{DoxyCode}

\DoxyCodeLine{\$pdf\ =\ App::make(\textcolor{stringliteral}{'dompdf.wrapper'});}

You can chain the methods\+:

\DoxyCodeLine{\textcolor{keywordflow}{return}\ Pdf::loadFile(public\_path().\textcolor{stringliteral}{'/myfile.html'})-\/>save(\textcolor{stringliteral}{'/path-\/to/my\_stored\_file.pdf'})-\/>stream(\textcolor{stringliteral}{'download.pdf'});}

\end{DoxyCode}


If you need the output as a string, you can get the rendered \doxylink{class_p_d_f}{PDF} with the output() function, so you can save/output it yourself.
