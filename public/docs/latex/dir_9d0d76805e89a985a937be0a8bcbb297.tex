\hypertarget{Readme.md_autotoc_md2322}{}\doxysubsection{\texorpdfstring{Data\+Tables}{Data\+Tables}}\label{Readme.md_autotoc_md2322}
This package contains distribution files for the \href{https://datatables.net}{\texttt{Data\+Tables table enhancement library}}. Only the core software for this library is contained in this package -\/ to be correctly styled, a styling package for \doxylink{class_data_tables}{Data\+Tables} must also be included (e.\+g. default, Bootstrap, Bulma, Foundation or others) -\/ please see the \href{https://datatables.net/manual/installation\#Node.js-/-NPM}{\texttt{npm installation documentation on the Data\+Tables site}} for full details.

\doxylink{class_data_tables}{Data\+Tables} is a table enhancing library which adds features such as paging, ordering, search, scrolling and many more to a static HTML page. A comprehensive API is also available that can be used to manipulate the table.
