\doxysubsection{Detailed Description}
This class represents a {\ttfamily Rule\+Set} constrained by a {\ttfamily Selector}.

It contains an array of selector objects (comma-\/separated in the CSS) as well as the rules to be applied to the matching elements.

Declaration blocks usually appear directly inside a {\ttfamily Document} or another {\ttfamily CSSList} (mostly a {\ttfamily Media\+Query}).
