\doxysubsubsection*{vite-\/plugin-\/full-\/reload}



Automatically reload the page when files are modified



\href{https://www.npmjs.com/package/vite-plugin-full-reload}{\texttt{ }} \href{https://github.com/ElMassimo/vite-plugin-full-reload/blob/main/LICENSE.txt}{\texttt{ }}

\hypertarget{README.md_autotoc_md7984}{}\doxysubsubsection{\texorpdfstring{Why? ü§î}{Why? ü§î}}\label{README.md_autotoc_md7984}
When using {\itshape \mbox{[}\doxylink{class_vite}{Vite} Ruby\mbox{]}}, I wanted to see changes to server-\/rendered layouts and templates without having to manually reload the page.

Also, in {\itshape \mbox{[}JS From Routes\mbox{]}} path helpers are generated when Rails reload is triggered.

Triggering a page reload when {\ttfamily config/routes.\+rb} is modified makes the DX very smooth.\hypertarget{README.md_autotoc_md7985}{}\doxysubsubsection{\texorpdfstring{Installation üíø}{Installation üíø}}\label{README.md_autotoc_md7985}

This is useful to trigger a page refresh for files that are not being imported, such as server-\/rendered templates.

To see which file globbing options are available, check \href{https://github.com/micromatch/picomatch\#globbing-features}{\texttt{picomatch}}.\hypertarget{README.md_autotoc_md7987}{}\doxysubsubsection{\texorpdfstring{Configuration ‚öôÔ∏è}{Configuration ‚öôÔ∏è}}\label{README.md_autotoc_md7987}
The following options can be provided\+:


\begin{DoxyItemize}
\item {\ttfamily root}

Files will be resolved against this directory.

{\bfseries{Default\+:}} {\ttfamily process.\+cwd()}


\begin{DoxyCode}{0}
\DoxyCodeLine{js}
\DoxyCodeLine{\ FullReload('config/routes.rb',\ \{\ root:\ \_\_dirname\ \}),}

\end{DoxyCode}

\item {\ttfamily delay}

How many milliseconds to wait before reloading the page after a file change. It can be used to offset slow template compilation in Rails.

{\bfseries{Default\+:}} {\ttfamily 0}


\begin{DoxyCode}{0}
\DoxyCodeLine{FullReload('app/views/**/*',\ \{\ delay:\ 100\ \})}

\end{DoxyCode}

\item {\ttfamily always}

Whether to refresh the page even if the modified HTML file is not currently being displayed.

{\bfseries{Default\+:}} {\ttfamily true}


\begin{DoxyCode}{0}
\DoxyCodeLine{FullReload('app/views/**/*',\ \{\ always:\ false\ \})}

\end{DoxyCode}
