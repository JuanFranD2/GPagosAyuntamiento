\doxysubsection{Detailed Description}
Convenience class to create a \doxylink{class_context}{Context} for Doc\+Blocks when not using the Reflection \doxylink{class_component}{Component} of php\+Documentor.

For a Doc\+Block to be able to resolve types that use partial namespace names or rely on namespace imports we need to provide a bit of context so that the Doc\+Block can read that and based on it decide how to resolve the types to Fully Qualified names.
