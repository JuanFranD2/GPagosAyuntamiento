\item All classes uses {\ttfamily \doxylink{namespace_masterminds}{Masterminds}} namespace.
\item All public static methods has been removed from {\ttfamily HTML5} class and the general API to access the HTML5 functionalities has changed.

Before\+: \begin{DoxyVerb}$dom = \HTML5::loadHTML('<html>....');
\HTML5::saveHTML($dom);
\end{DoxyVerb}


After\+: \begin{DoxyVerb}use Masterminds\HTML5;

$html5 = new HTML5();

$dom = $html5->loadHTML('<html>....');
echo $html5->saveHTML($dom);
\end{DoxyVerb}
 
\end{DoxyItemize}
