\doxysubsection{Detailed Description}
This class is a container for individual \textquotesingle{}Rule\textquotesingle{}s.

The most common form of a rule set is one constrained by a selector, i.\+e., a {\ttfamily Declaration\+Block}. However, unknown {\ttfamily At\+Rule}s (like {\ttfamily @font-\/face}) are rule sets as well.

If you want to manipulate a {\ttfamily Rule\+Set}, use the methods {\ttfamily add\+Rule(Rule \$rule)}, {\ttfamily get\+Rules()} and {\ttfamily remove\+Rule(\$rule)} (which accepts either a {\ttfamily Rule} or a rule name; optionally suffixed by a dash to remove all related rules).

remove\+Rule (\begin{DoxyParamCaption}\item[{}]{\$m\+Rule}{}\end{DoxyParamCaption})}

Removes a rule from this Rule\+Set. This accepts all the possible values that {\ttfamily get\+Rules()} accepts.

If given a Rule, it will only remove this particular rule (by identity). If given a name, it will remove all rules by that name.

Note\+: this is different from pre-\/v.\+2.\+0 behaviour of PHP-\/\+CSS-\/\+Parser, where passing a Rule instance would remove all rules with the same name. To get the old behaviour, use {\ttfamily remove\+Rule(\$o\+Rule-\/\texorpdfstring{$>$}{>}get\+Rule())}.
