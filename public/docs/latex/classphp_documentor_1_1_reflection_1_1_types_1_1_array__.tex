\doxysubsection{Detailed Description}
Represents an array type as described in the PSR-\/5, the PHPDoc Standard.

An array can be represented in two forms\+:


\begin{DoxyEnumerate}
\item Untyped ({\ttfamily array}), where the key and value type is unknown and hence classified as \textquotesingle{}Mixed\+\_\+\textquotesingle{}.
\item Types ({\ttfamily string\mbox{[}\mbox{]}}), where the value type is provided by preceding an opening and closing square bracket with a type name.
\end{DoxyEnumerate}
