\doxysubsection{Detailed Description}
\doxysubsection*{PHPDoc Parser for PHPStan}



\href{https://github.com/phpstan/phpdoc-parser/actions}{\texttt{}} \href{https://packagist.org/packages/phpstan/phpdoc-parser}{\texttt{}} \href{https://choosealicense.com/licenses/mit/}{\texttt{}} \href{https://phpstan.org/}{\texttt{}} 

This library {\ttfamily phpstan/phpdoc-\/parser} represents PHPDocs with an AST (Abstract Syntax Tree). It supports parsing and modifying PHPDocs.

For the complete list of supported PHPDoc features check out PHPStan documentation. PHPStan is the main (but not the only) user of this library.


\begin{DoxyItemize}
\item \href{https://phpstan.org/writing-php-code/phpdocs-basics}{\texttt{PHPDoc Basics}} (list of PHPDoc tags)
\item \href{https://phpstan.org/writing-php-code/phpdoc-types}{\texttt{PHPDoc Types}} (list of PHPDoc types)
\item \href{https://phpstan.github.io/phpdoc-parser/2.1.x/namespace-PHPStan.PhpDocParser.html}{\texttt{phpdoc-\/parser API Reference}} with all the AST node types etc.
\end{DoxyItemize}

This parser also supports parsing \href{https://github.com/doctrine/annotations}{\texttt{Doctrine Annotations}}. The AST nodes live in the \href{https://phpstan.github.io/phpdoc-parser/2.1.x/namespace-PHPStan.PhpDocParser.Ast.PhpDoc.Doctrine.html}{\texttt{PHPStan\textbackslash{}\+Php\+Doc\+Parser\textbackslash{}\+Ast\textbackslash{}\+Php\+Doc\textbackslash{}\+Doctrine namespace}}.\hypertarget{README.md_autotoc_md10128}{}\doxysubsubsection{\texorpdfstring{Installation}{Installation}}\label{README.md_autotoc_md10128}
