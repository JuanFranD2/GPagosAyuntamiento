\doxysubsection{Detailed Description}
This is the most generic container available. It can contain {\ttfamily Declaration\+Block}s (rule sets with a selector), {\ttfamily Rule\+Set}s as well as other {\ttfamily CSSList} objects.

It can also contain {\ttfamily Import} and {\ttfamily Charset} objects stemming from at-\/rules.
