generate (\begin{DoxyParamCaption}\item[{string}]{\$name}{, }\item[{array}]{\$parameters}{ = {\ttfamily \mbox{[}\mbox{]}}, }\item[{int}]{\$reference\+Type}{ = {\ttfamily self\+:\+:ABSOLUTE\+\_\+PATH}}\end{DoxyParamCaption})}

Generates a \doxylink{class_u_r_l}{URL} or path for a specific route based on the given parameters.

Parameters that reference placeholders in the route pattern will substitute them in the path or host. Extra params are added as query string to the \doxylink{class_u_r_l}{URL}.

When the passed reference type cannot be generated for the route because it requires a different host or scheme than the current one, the method will return a more comprehensive reference that includes the required params. For example, when you call this method with \$reference\+Type = ABSOLUTE\+\_\+\+PATH but the route requires the https scheme whereas the current scheme is http, it will instead return an ABSOLUTE\+\_\+\+URL with the https scheme and the current host. This makes sure the generated \doxylink{class_u_r_l}{URL} matches the route in any case.

If there is no route with the given name, the generator must throw the Route\+Not\+Found\+Exception.

The special parameter \+\_\+fragment will be used as the document fragment suffixed to the final \doxylink{class_u_r_l}{URL}.
