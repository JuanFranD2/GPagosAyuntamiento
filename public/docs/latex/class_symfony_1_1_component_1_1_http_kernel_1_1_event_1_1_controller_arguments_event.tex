\doxysubsection{Detailed Description}
Allows filtering of controller arguments.

You can call get\+Controller() to retrieve the controller and get\+Arguments to retrieve the current arguments. With set\+Arguments() you can replace arguments that are used to call the controller.

Arguments set in the event must be compatible with the signature of the controller.
