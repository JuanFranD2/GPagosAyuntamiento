\doxysubsection{Detailed Description}
A {\ttfamily CSSBlock\+List} is a {\ttfamily CSSList} whose {\ttfamily Declaration\+Block}s are guaranteed to contain valid declaration blocks or at-\/rules.

Most {\ttfamily CSSList}s conform to this category but some at-\/rules (such as {\ttfamily @keyframes}) do not.
